# WiFi Talker Signaling Server

Простой WebSocket сервер для обмена сигналами между устройствами в приложении WiFi Talker.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Сервер запустится на порту 8080 (или на порту, указанном в переменной окружения PORT).

## Использование

1. Узнайте IP адрес вашего компьютера в локальной сети
2. В приложении Android измените URL сервера на `ws://<ваш-IP>:8080`
3. Запустите сервер
4. Подключитесь с двух устройств, используя один и тот же ID комнаты

## Пример

Если IP вашего компьютера `192.168.1.100`, то URL будет:
```
ws://192.168.1.100:8080
```

## Протокол

Сервер принимает WebSocket соединения по пути:
```
ws://<host>:<port>/<room-id>?initiator=true|false
```

Сообщения передаются в формате JSON:
- `offer` - SDP offer
- `answer` - SDP answer  
- `ice-candidate` - ICE candidate
